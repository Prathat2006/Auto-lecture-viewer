        • Week-4

          • Activity With Google Colab Assignment
            To do: View To do: Make a submission
            Opened: Saturday, 1 March 2025, 10:22 AM
            Due: Monday, 3 March 2025, 9:00 PM

            "Cargo Loading Problem"
            Problem:
            Imagine you're working as a logistics coordinator for a shipping company. You have several cargo containers, each with different types of items, and your task is to calculate the total weight of the items in each container. Each type of item has a specific weight, and each container holds a different number of items of each type.

            You are given the following:

            The weights of each item type (a vector).
            The number of items of each type in each container (a matrix, where each row represents a different container).
            Your task is to:

            Create a vector that represents the weights of each item.
            Create a matrix that represents the number of items of each type in different containers.
            Multiply the vector (weights) by the matrix (number of items) to calculate the total weight of each container.
            Instructions:
            Create a vector representing the weights of 4 different types of items.
            Create a matrix where each row represents a container, and each column represents how many of each item type is in that container (4 containers, 4 item types).
            Multiply the vector and the matrix using Numpy to find the total weight in each container.
            Print the total weights of each container.
            Example Data:
            Item weights (vector):
            Item 1 weighs 10 kg, Item 2 weighs 20 kg, Item 3 weighs 15 kg, Item 4 weighs 30 kg.

            Containers' item quantities (matrix):

            Container 1: 2 items of type 1, 1 item of type 2, 3 items of type 3, 1 item of type 4.
            Container 2: 1 item of type 1, 4 items of type 2, 2 items of type 3, 1 item of type 4.
            Container 3: 3 items of type 1, 2 items of type 2, 1 item of type 3, 4 items of type 4.
            Container 4: 5 items of type 1, 1 item of type 2, 2 items of type 3, 3 items of type 4.

            Upload the screenshot of your program and output. Use Google CoLab for this exercise.
